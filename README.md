# text-search-image
This sample is demonstrating how to use text to search an image. 

## How to run it?

**1. Install the requirements**:

    $python3 -m venv openvino_env

    $source openvino_env/bin/activate

    $pip install -r requirements.txt

**2. Export the models**:

    $python3 export.py

**3. Run application**:

    $python3 app.py
